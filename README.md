# 🔐 Base64 Encoder/Decoder

A free, open-source online Base64 encoder and decoder. Convert between text and Base64, or encode files — all in your browser. No server, no signup.

**Live demo:** [vaultool.com/tools/base64-converter.html](https://vaultool.com/tools/base64-converter.html)

## ✨ Features

- **Text Encoding** — Convert plain text to Base64
- **Text Decoding** — Convert Base64 back to readable text
- **File Encoding** — Encode any file to Base64 (up to 5MB)
- **UTF-8 Support** — Handles non-ASCII characters correctly
- **Swap** — Instantly swap input and output
- **Copy & Download** — One-click copy or download results
- **Privacy First** — 100% client-side, your data never leaves the browser
- **Zero Dependencies** — Single HTML file, no build step required

## 🚀 Quick Start

Just open `index.html` in any modern browser. No installation needed.

## 🛠️ Tech Stack

- Zero dependencies
- No build tools required
